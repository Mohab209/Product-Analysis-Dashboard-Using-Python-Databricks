# 📊 Product Analysis Dashboard (Python + Databricks + Streamlit)

An end-to-end Product Analytics Dashboard built using Python, Databricks, and Streamlit.  
The project focuses on real-time data extraction, and interactive visualization with automated refresh and cloud deployment.

---

## 📌 Project Overview

This project demonstrates a complete analytics pipeline from data source to deployed dashboard:

- Connecting to a Databricks SQL Warehouse  
- Extracting data using databricks_sql_connector  
- Loading data into Pandas DataFrames   
- Building an interactive dashboard using Streamlit  
- Deploying the application on Streamlit Cloud  
- Securing credentials using Streamlit Secrets  
- Automating data refresh for real-time insights  
- Customizing Streamlit UI (layout, colors, and design)
